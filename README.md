# 🛒 ShopVerse

ShopVerse is a modern Amazon-inspired e-commerce frontend built using React, TypeScript, Vite, Tailwind CSS, and TanStack Start. It provides a full shopping UI experience including product browsing, filtering, cart management, wishlist, and checkout flow.

---

## 🚀 Features

- 🏠 Home page with banners and category sections  
- 🔍 Product listing with filters (price, rating, category, brand)  
- 📦 Product detail page with image gallery and reviews UI  
- 🛒 Cart system with quantity updates and price summary  
- ❤️ Wishlist functionality  
- 💳 Multi-step checkout UI flow  
- 🔎 Search with suggestions and recent searches  
- ⏳ Skeleton loading states for better UX  
- 📱 Fully responsive mobile-first design  

---

## 🧑‍💻 Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- TanStack Start
- Zustand (state management)
- Framer Motion (animations)
- Shadcn UI
